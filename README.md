# 🌦️ Weather & AQI App

A simple React app to check **weather** and **air quality index (AQI)** by entering any city name.

---

## 🚀 Features

- 🌤️ Real-time weather info (temperature, humidity, wind, etc.)
- 🌫️ Air Quality Index (AQI) data
- 🔍 Search by city name
- 📸 Background images
- 🧪 Built using React
